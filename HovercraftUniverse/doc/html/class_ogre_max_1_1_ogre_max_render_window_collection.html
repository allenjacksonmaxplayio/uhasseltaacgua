Detailed Description

template<class COLLECTION, class ITERATOR>
class OgreMax::OgreMaxRenderWindowCollection< COLLECTION, ITERATOR >

Accesses a render window collection (std::list, std::vector, std::set, and so on) Example usage with a non-const list: OgreMaxRenderWindowCollection<std::list<Ogre::RenderWindow*>&, std::list<Ogre::RenderWindow*>::iterator> renderWindows(myRenderWindowList); Example usage with a const vector: OgreMaxRenderWindowCollection<const std::vector<Ogre::RenderWindow*>&, std::vector<Ogre::RenderWindow*>::const_iterator> renderWindows(myRenderWindowVector);
