bool OgreMax::Types::LoadedRenderTexture::GetReferencePosition ( Ogre::Vector3 &  position  )  const [inline]

Gets the 'reference' position, which is used when updating a cube map render texture. The preferred position is that of the render object. If there is no render object, the reference camera is used. If there is no camera, the zero vector is used.

Parameters:
position [out] - The position. If there is no reference object, this is set to zero.
Returns:
Returns true if there was a reference object to use, false otherwise.
