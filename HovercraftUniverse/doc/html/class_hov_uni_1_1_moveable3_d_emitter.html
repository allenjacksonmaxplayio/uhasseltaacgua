Detailed Description

A virtual class representing a moveable 3D sound emitter. Objects that want to emmit sound in a three dimensional space must extend this class and implement missing methods.

Author:
Nick De Frangh

Ogre::Vector3 **  orientation 
) [protected, pure virtual]

This function will be called every cycle to get updates for the position, velocity and orientation. This is needed to calculate the 3D sound in a realistic way.

Parameters:
position Position in 3D space of the event. Specifying 0 / null will ignore this parameter.
velocity Velocity in 'distance units per second' in 3D space of the event. See remarks. Specifying 0 / null will ignore this parameter
orientation Orientation of the event sound cone. Only used if the event has a cone specified to determine cone detection, otherwise just specify 0 / null. Specifying 0 / null will ignore this parameter.
