Read this!
==========

1) Be sure to change your nickname by editing 
   HovercraftUniverse.ini and replacing "UnnamedPlayer"
   with your own name.

2) Launch Hovercraft Universe using the shortcuts on 
   your desktop, or by running HovercraftUniverse.exe
   in the installation directory!

   TIP: Make sure you have administrator rights! If not,
        install the game in My Documents or another 
        personal folder.
Have fun!


More Info
=========

Visit our web site at http://code.google.com/p/uhasseltaacgua/
for new versions and more information about the game!


Troubleshooting
===============
- Missing "d3dx9_42.dll" file

   Install DirectX using the DirectX setup file on the cd, 
   or download the DirectX End-User Runtime directly from
   Microsoft: 

   http://www.microsoft.com/downloads/details.aspx?FamilyID=2da43d38-db71-4c1b-bc6a-9b6652cd92a3&displaylang=en
